# Counter-Spy Private Docs Staging Repo

This repository is a markdown-only staging copy of the Counter-Spy.ai documentation set.

Purpose:
- hold product and technical guidance separately from the primary engineering repo
- stay private until the codebase and publish posture are finalized
- make it easier to promote selected docs to a future public-facing repository

Current contents:
- README
- SBOM
- Technical Specification
- Operations Guide
- selected docs from `docs/`

Not included:
- application source code
- backend runtime code
- infrastructure templates beyond documentation references
- local secrets or environment files

Suggested next step:
- create a private GitHub repository for this folder
- push this repo as the documentation holding area
- later decide whether to keep it private, make it public, or split public/private docs further
